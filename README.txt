# HR Analytics Dashboard

A professional HR Analytics Dashboard created using Power BI to analyze workforce trends, employee demographics, remote work statistics, and hiring patterns.

## 📌 Project Overview

This dashboard provides insights into:
- Total employees
- Remote vs HQ workforce
- Hiring trends
- Gender distribution
- Department analysis
- State-wise workforce analysis
- Race distribution

## 📊 Dashboard Features

- KPI Cards
- Interactive Filters
- Hiring Trend Analysis
- Donut Chart Visualization
- Bar Chart Analysis
- Key Business Insights

## 🛠 Tools & Technologies Used

- Power BI
- SQL
- Power Query
- DAX
- Data Visualization
